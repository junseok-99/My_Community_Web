# My_Community_Web
PHP를 사용해 만든 나만의 커뮤니티 사이트 입니다.

<h2>1.Intro</h1>
<p><b>- 홈페이지에 첫 접속한 사진이다. 기본적으로 회원가입, ID/PW찾기, 로그인, 최근게시물 보여주기, 메뉴 바 기능을 포함한다.</b></p>
